Last Stand

Project Details

Tools Used: Unreal Engine (Blueprints, 3D Models, Materials & Textures)

Duration: 3 months (full-time)

Restrictions: My first game made with Unreal Engine, developed in a short time frame.

Description: Last Stand was my first project using Unreal Engine and my first time creating a game from scratch with it. This was part of my university coursework, where I had to design a playable game that demonstrates both technical and creative skills. I decided to create a first-person zombie survival shooter, where the player has to fight through waves of zombies in a post-apocalyptic environment. The game features increasing difficulty with each wave, keeping the player under constant pressure and creating a tense survival experience. Time was limited during this project, so I had to quickly adapt to using Unreal Engine while also learning key concepts like level design, game mechanics, and player interaction. Through this project, I developed a better understanding of game design and gained hands-on experience with building immersive gameplay. It also allowed me to push myself creatively while learning how to manage time and solve problems under pressure.
