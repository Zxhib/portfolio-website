Project Details

Tools Used: Unity (Game Development), C# (Programming), GitKraken (Version Control for Group Collaboration)

Duration: 2 months (full-time)

Restrictions: Limited time for multiplayer networking, which required optimising the game for smooth online gameplay.

Description: Blaze Ball is a local multiplayer football game created collaboratively by me and two other team members using Unity, C#, and GitKraken for version control. Each of us contributed to different parts of the game, and my contributions were focused on both game mechanics and user interface design. The game is designed for 4 players and is set on a small pitch surrounded by trees, stones, and small hills. One unique feature is that there are invisible walls around the pitch, so the ball cannot go out of play like it would in real football. These invisible walls can be an advantage or disadvantage, depending on whether a team is attacking or defending.

My tasks included developing the following scripts:

PlayerPickUp script – Handling player interactions with the ball.

MainMenu script – Creating the main menu interface and functionality.

GameTimer script – Managing the game's time system.

GameOverScreen script – Displaying the game over screen when the game finishes.

PauseMenu script – Allowing players to pause and resume the game.

Additionally, I worked on designing the fonts, buttons, and overall look for all the menus to ensure a user-friendly and visually appealing experience.
