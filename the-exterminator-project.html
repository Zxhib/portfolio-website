The Exterminator

Project Details

Tools Used: Unity (Game Development), Itch.io (2D Models), OpenGameArt (Audio/Sound Effects)

Duration: 3 months (full-time)

Restrictions: First-time Unity user with no prior experience.

Description: The Exterminator is a 2D game I developed as my first-ever project using Unity. As it was my initial experience with game development, I faced the challenge of understanding how a game engine works and learning how to manipulate its features to create a polished gameplay experience.

In the game, players control a character with an oversized head whose objective is to eliminate a mouse to collect keys and coins. The character has three lives, and upon death, the game restarts from the last activated checkpoint, adding a strategic layer to the gameplay. This project helped me build a strong foundation in game mechanics, level design, and Unity’s core functionalities.
